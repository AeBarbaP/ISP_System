# ISP_System – Internet Service Provider Management System

## Overview

ISP_System is a web application developed to manage clients, services, and operational processes for an Internet Service Provider.

The system allows efficient administration of customers and service data through a centralized platform.

## Features

* Customer management
* Service management
* Administrative panel
* Database integration
* Secure access system

## Technologies Used

* PHP
* MySQL
* JavaScript
* HTML
* CSS
* Bootstrap
